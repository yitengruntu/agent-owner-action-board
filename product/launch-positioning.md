# Launch Positioning

## Tagline

Know what your local AI agent needs from you next.

## Short Description

Agent Owner Action Board is a local report for autonomous agent projects. It turns blockers into explicit owner actions, so the agent knows when to stop and the owner knows exactly what to provide next.

## Audience

People experimenting with Codex, Claude Code, Cursor, local cron jobs, or custom agent loops.

## CTA

Inspect the local demo.

## Objections

### "I can just check the terminal."

The terminal shows what happened. The board shows what the owner needs to do next.

### "This should be built into the agent."

Probably. Until then, a repo-readable report is enough to test whether the workflow matters.

### "I do not want another dashboard."

The first version is a static local HTML report that can live inside a project repo.
